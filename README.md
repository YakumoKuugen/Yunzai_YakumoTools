# 安装
进入Yunzai根目录，运行下述指令：

使用github
```bash
git clone --depth=1 https://github.com/YakumoKuugen/Yunzai_YakumoTools.git ./plugins/YakumoTools
```

安装后重启Yunzai即可使用。

# 功能
